# 2026 全球 AI Native 组织全维度调研 · 交互式全景报告

> 从"大语言模型"向"全局智能体生态"的范式跃迁——深度解码全球九大顶尖 AI 组织的技术演进、组织密码与未来蓝图。

---

## 项目概述

本项目将《2026 全球 AI Native 组织全维度调研与未来组织建设全景报告》转化为一个交互式单页网页应用，以可视化、结构化的方式呈现报告全部内容，便于阅读、理解与分享。

## 涵盖组织

| 阵营 | 组织 |
|------|------|
| 海外 | OpenAI · Anthropic · Google DeepMind · xAI |
| 中国 | DeepSeek · MiniMax · 月之暗面 (Moonshot AI) · 字节跳动 Seed · 阿里通义千问 |

## 报告结构

网页共包含 **7 个核心模块**：

1. **引言** — 范式跃迁的历史背景与报告定位
2. **九家总览** — 全组织横向数据表（估值、人数、技术路线、旗舰模型、开源策略）
3. **海外前沿巨头** — OpenAI / Anthropic / DeepMind / xAI 深度卡片
4. **中国创新者** — DeepSeek / MiniMax / Moonshot / 字节 Seed / 阿里 Qwen 深度卡片
5. **跨组织对比** — 三标签页切换：组织矩阵 · 东西方差异 · 薪酬可视化
6. **激励机制拆解** — 六大组织的薪酬、股权、文化激励详细对比
7. **核心发现** — 三大演化逻辑（选拔 · 激励 · 人机交互范式）

## 交互功能

- **固定导航栏** — 滚动时自动高亮当前章节，点击快速跳转
- **可展开卡片** — 点击组织卡片展开/收起详细内容
- **标签页切换** — 跨组织对比模块支持三视图无缝切换
- **薪酬条形图动画** — 进入视口或切换标签时自动触发动画
- **滚动渐入** — 各模块内容随滚动逐步显现
- **导航自动隐藏** — 向下滚动时导航栏收起，向上滚动时恢复

## 技术实现

- **纯静态单文件** — 单个 `.html` 文件，零依赖，零构建步骤
- **原生技术栈** — HTML5 + CSS3 + Vanilla JavaScript，无任何框架
- **响应式设计** — 适配桌面端与移动端
- **字体** — Noto Sans SC（中文）+ Space Mono（数据/标签）+ DM Sans（英文正文）
- **配色方案** — 深色科技主题，蓝/绿/紫/橙四色辅助体系

## 使用方式

直接在浏览器中打开 `2026_AI_Native_全景报告.html` 即可，无需安装任何依赖或启动服务器。

```bash
# macOS
open 2026_AI_Native_全景报告.html

# Windows
start 2026_AI_Native_全景报告.html

# Linux
xdg-open 2026_AI_Native_全景报告.html
```

## 数据来源
